# To_do_list
Todo app built with React using REST API 
# Basic Features
+ User can login and signup to save  todos
+ User can create a Todo
+ User can see their Todos
+ User can mark the Todo as done
+ User can delete the Todo from the list
+ User can see the daytime
## Built with
- ReactJS
- REST API
